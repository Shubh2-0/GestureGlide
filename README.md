# GestureGlide 

Welcome to GestureGlide, a captivating Rock-Paper-Scissors game implemented using Python and OpenCV. Play against an AI opponent and use hand gestures to determine your move! 锔

## Features 

- Real-time hand tracking using OpenCV.
- Play against an AI that randomly selects Rock, Paper, or Scissors.
- Interactive timer to track game duration 憋.
- Visual representation of your move and AI's move .
- Keeps track of player and AI scores .

## Installation 锔

 Clone the repository:

bash
``
   git clone https://github.com/Shubh2-0/gesture-glide.git
   cd gesture-glide
``

 Install the required libraries:

bash
``
pip install opencv-python cvzone
``

 Place your background image and move images in the resources/ directory.

## Usage 

达 Run the game script:
``
python game.py
``
达  Press the 'x' key to start the game after the webcam feed is displayed.

达  Show your hand gesture to select Rock, Paper, or Scissors when the timer reaches 3 seconds.

达  The AI will make its move simultaneously, and the winner of the round will be displayed.

## Controls 

 Press 'x': Start the game.

 Gesture your hand to select Rock, Paper, or Scissors.
